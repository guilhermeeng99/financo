# Package Name

The `app_core` package serves as the central foundation for building robust Flutter applications by providing essential features and utilities that streamline application architecture and development. 

## Features

- Core utilities and components for Flutter application development.
- Integration with `flutter_rearch` for streamlined reactive architecture.
- Interfacing with `rearch` to enable enhanced state management solutions.

## Installation

To use the `app_core` package, add it as a dependency in your `pubspec.yaml` file:

```yaml
dependencies:
  flutter:
    sdk: flutter
  app_core:
    path: packages/app_core
```

## Dependencies

The package relies on the following dependencies to deliver its features:

- `flutter_rearch`: Facilitates reactive architecture capabilities in Flutter apps.
- `rearch`: Provides tools for effective state management and architectural patterns.
- `flutter`: The core Flutter SDK allowing the package to integrate seamlessly with Flutter apps.

## Dev Dependencies

- `flutter_test`: Supports testing Flutter applications.
- `very_good_analysis`: Assists in maintaining high code quality through linting and analysis.
